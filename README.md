🥤 Coca-Cola US Sales Dashboard

An interactive Excel dashboard analyzing Coca-Cola beverage sales across the United States for 2022–2023.

📊 Overview

Built entirely in Excel with raw data, pivot-based analysis, and a visual dashboard — this project explores retail sales performance across regions, brands, and time periods.

🗂️ Workbook Structure

SheetDescriptionDataRaw transaction data — 8000+ rows covering retailer, location, brand, pricing, units sold, revenue, and operating marginAnalyzePivot tables summarizing KPIs by brand, retailer, and quarterly trendsDashboardVisual summary with slicers for interactive filtering

📦 Key KPIs

Metric- Total Sales-$12M📦, Total Units Sold-24.8M📈, Operating Profit-$4.72M💵, Avg Price per Unit-$0.45📉.

🔍 Highlights

Top Brand: Coca-Cola ($2.77M in sales)
Top Retailer: West Soda ($3.24M in sales)
Best Quarter: Q3 2023 — $2.81M in sales with 43.5% avg margin
Regions Covered: Northeast, Southeast, South, Midwest, West

🏪 Retailers & Brands

Retailers: Amazon, BevCo, FizzyCo, Target, Walmart, West Soda
Brands: Coca-Cola, Diet Coke, Sprite, Fanta, Powerade, Dasani Water

🛠️ Tools Used

Microsoft Excel (Pivot Tables, Slicers, Combo Chart)
